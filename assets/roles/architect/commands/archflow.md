Architect 工作流调度器。

## 用法

```
/archflow "设计用户权限系统架构"
```

## 工作流阶段

| 阶段 | 技能 | 描述 |
|------|------|------|
| research | arch-research | 方案调研 |
| design | arch-design | 架构设计 |
| review | arch-review | 评审 |
| guide | arch-guide | 落地指导 |

## 相关命令

- `/archflow-research` — 快速技术调研
- `/archflow-review` — 启动架构评审
