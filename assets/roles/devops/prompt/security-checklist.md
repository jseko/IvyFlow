# 部署安全检查清单

## 密钥管理

- [ ] 密钥不硬编码在代码中
- [ ] 使用密钥管理服务（Vault / AWS Secrets Manager）
- [ ] CI/CD 中使用环境变量注入

## 网络策略

- [ ] 最小权限原则（仅开放必要端口）
- [ ] 内部服务不暴露公网
- [ ] API 有速率限制

## 访问控制

- [ ] 生产环境访问需要审批
- [ ] SSH 密钥定期轮换
- [ ] 操作审计日志已开启

## 镜像安全

- [ ] 使用官方基础镜像
- [ ] 镜像已扫描漏洞
- [ ] 不使用 latest 标签

## 备份恢复

- [ ] 数据库定期备份
- [ ] 备份恢复流程已测试
- [ ] RTO/RPO 已定义
