DevOps 工作流调度器。

## 用法

```
/devopsflow "为项目搭建 CI/CD"
```

## 工作流阶段

| 阶段 | 技能 | 描述 |
|------|------|------|
| env | devops-env | 环境规划 |
| cicd | devops-cicd | CI/CD 搭建 |
| deploy | devops-deploy | 部署 |
| monitor | devops-monitor | 监控 |
| alert | devops-alert | 告警 |

## 相关命令

- `/devopsflow-deploy` — 快速部署
- `/devopsflow-monitor` — 查看监控状态
