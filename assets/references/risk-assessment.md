# 风险评估模板

## 风险矩阵

| 风险 | 影响 | 概率 | 等级 | 缓解措施 | 残余风险 |
|------|------|------|------|---------|---------|
| | 高/中/低 | 高/中/低 | | | |

## 影响等级

- **高**：核心功能不可用、数据丢失、安全漏洞
- **中**：次要功能异常、性能下降、用户体验受损
- **低**：UI 瑕疵、文案错误、非关键路径问题

## 概率等级

- **高**：几乎必然发生（>70%）
- **中**：可能发生（30-70%）
- **低**：不太可能发生（<30%）

## 缓解措施模板

1. 预防措施（降低概率）
2. 减轻措施（降低影响）
3. 应急预案（发生后响应）
