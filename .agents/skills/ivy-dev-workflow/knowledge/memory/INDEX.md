# 记忆索引

> 记忆引擎检索入口。每次功能实现归档时，自动追加一行摘要记录。
> 创建时间：2026-05-29（v3.2）

---

## 检索策略

- **触发条件**：用户需求中出现技术标签且匹配 ≥ 2 个
- **匹配方式**：标签交集法（用户需求标签 ∩ 记忆行标签）
- **返回结果**：匹配的记忆行按日期倒序排列，最多返回 5 条

## 去重策略

- 同一主题多条记忆 → 按日期保留最新 3 条
- 内容相似度 > 80% → 合并为 1 条（保留最新日期，合并标签）
- 冲突标签 → 标记为"⚠️ 待确认"，由用户判断

## 过期策略

- 最后访问距今 > 6 个月 → 移至 `archive/`
- 归档前压缩：保留摘要 + 标签，原文删除
- archive/ 中的记忆不参与检索
